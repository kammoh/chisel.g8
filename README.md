# chisel.g8

A minimal [Chisel] template [Giter8](http://www.foundweekends.org/giter8/) template using [Mill](https://com-lihaoyi.github.io/mill/mill/Intro_to_Mill.html) and 
based on [com-lihaoyi/mill-scala-hello.g8](https://github.com/com-lihaoyi/mill-scala-hello.g8).

## Getting Started

_Usage with Giter8_

```sh
g8 kammoh/chisel.g8
```

_Usage with Mill_

```sh
mill init kammoh/chisel.g8
```
